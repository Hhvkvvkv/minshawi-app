# مجلس المنشاوي - ديوان التلاوات

تطبيق Android لسماع تلاوات الشيخ محمد صديق المنشاوي رحمه الله.

## الميزات
- تلاوات إذاعية (597 تلاوة)
- محافل خارجية (307 تلاوة)
- مصحف مجود (194 تلاوة)
- مصحف مرتل (114 تلاوة)
- مساعدة ذكية (Shazam للتلاوات)

## البناء
1. `npm install`
2. `npx expo prebuild --platform android`
3. `cd android && ./gradlew assembleRelease`

## الرخصة
جميع الحقوق محفوظة لصالح المجلس الأعلى 2026
